    The most popular and not so popular JS libraries out there

    Everyone is using javascript libraries nowadays but linking to several sources might be annoying. There's Google Ajax API and other services but they just choose what's better for you and host only most popular libraries.

    With OpenCDN.me you have them all in one place. That's neat right?

    Always up to date!

    Be sure that the script you are using is always up to date and you have the latest version available. No need to check library websites or update the links in your code. OpenCDN.me will take care of that.
    All scripts are minified so your website should not gain too much additional weight.

    There is plenty to choose from! See?
